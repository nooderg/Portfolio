# Portfolio
Very not up to date, but sick design 🔥
This is my portfolio. And here's a guide on how to make it run on your machine

## Installation

```bash
git clone https://github.com/TheoFenique/Portfolio.git
```
```bash
cd Portfolio
```
```bash
npm i
```
```bash
npm start
```
